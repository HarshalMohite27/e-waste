# E-Waste Awareness Website frontend

A responsive frontend website created to spread awareness about electronic waste management and recycling.  
This project aims to encourage users to handle e-waste responsibly and protect the environment.

---

#  Features
- Informative sections about e-waste and recycling  
- Clean, modern, and responsive design  
- Easy navigation and visually appealing layout  
- Built using only frontend technologies  

---

# Technologies Used
- **HTML** – For page structure  
- **CSS** – For design and layout  
- **JavaScript** – For interactivity and dynamic content  

---

# How to View
Visit the live demo:  
https://harshalmohite27.github.io/e-waste/

---

# Purpose
To educate people about the dangers of improper e-waste disposal and promote sustainable recycling practices for a cleaner planet.
